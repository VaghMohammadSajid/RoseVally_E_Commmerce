# 🌹 RoseValley - E-commerce Website

## 📌 Overview
RoseValley is an **E-commerce platform** developed as a college project using **Java, JSP, Servlet, and Hibernate**.  
It provides a simple online shopping experience with authentication, product listing, and order management.

## 🚀 Features
- User Authentication  
- Product Catalog & Search  
- Shopping Cart  
- Order Placement & Tracking  
- Admin Panel for product management  

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Java, JSP, Servlet, Hibernate  
- **Database:** MySQL  

## ▶️ How to Run
1. Clone repo into Tomcat server directory.  
2. Configure MySQL database and update `hibernate.cfg.xml`.  
3. Build and deploy on Apache Tomcat.  
4. Access the app at `http://localhost:8080/RoseValley`.
